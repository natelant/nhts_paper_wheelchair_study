Trips

Activities

Activities with Tour Attributes

Using the function add_tour() each activity can be assigned to a daily activity pattern (DAP) which describes the type of tour or group of tours a person makes during the day. A tour is a collection of activities from when you leave home to when you return home. Often, people take more than one tour per day. Tours are classified as Mandatory (work or school), Non-Mandatory, and Home. For the purpose here, we label each tour as “W” for work, “S” for school, and “NM” for non-mandatory (everything else). Home trips will show up as NA for now.

Estimated Population and Counts of DAP Groups

This table shows the same distribution but of total counts from the list of persons NOT activities. The total number of persons is 23,906 in this set. This data has been filtered by age and by MSA size, as described previously. Using the weighted population estimates in the NHTS data, the probable total population has been summed in each group.

# this calculates the population in each group, based on the tour distribution
mytours %>%
  filter(Ability != "NA", 
         DAP != "NA") %>%
    # Calculate number of people in each combination of dap and ability.
    group_by(Ability, DAP) %>%
    summarise(population = sum(wtperfin)) %>%
    spread(Ability, population) %>%
  pander(caption = "Population of Each DAP Group")
