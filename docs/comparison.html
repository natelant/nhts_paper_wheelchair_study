American Community Survey

The ACS provides information on disability status and can be used to validate fidings from the NHTS. The tables can be found here.

NHTS Data on Disabilities

Different from the previous page, this table of distributions only compares the abled population and the disabled population. This table also includes an “Overall” column which shows the distribution of the total civilian population.

The variables and grouping of variables are designed to match the ACS variables (i.e. age, mode, education)
